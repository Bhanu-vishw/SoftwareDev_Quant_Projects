# Quantitative Finance Models and Trading Systems

This repository contains a collection of projects developed during my study of financial modeling and quantitative finance. The projects explore core concepts in derivatives pricing, trading system design, stochastic simulation, and financial analytics.

Implemented models include the Black–Scholes option pricing framework, Cox–Ross–Rubinstein binomial trees, Monte Carlo simulation techniques, and volatility modeling. The repository also contains components for trading system infrastructure, backtesting frameworks, and financial analytics tools built using Python.

Each project focuses on practical implementation of quantitative methods used in modern financial markets, with reproducible workflows and numerical experiments to analyze pricing behavior, risk metrics, and strategy performance.

# Repository Projects

## Brokerage API Analytics Pack

Description:
Tools for analyzing brokerage account data, trade history, and portfolio performance using Python-based data pipelines and financial analytics.

## Live Stock Trading Analytics

Description:
Real-time analytics framework for monitoring live trading strategies, tracking performance metrics, and evaluating portfolio risk in active equity positions.

## Backtesting Library

Description:
A modular backtesting framework for evaluating systematic trading strategies with support for transaction cost modeling and portfolio performance metrics.

## Blackjack Simulation

Description:
Monte Carlo simulation of blackjack strategies to analyze probability distributions, expected returns, and optimal decision strategies.

## Monte Carlo Simulations

Description:
Collection of stochastic simulation models used for financial modeling, probability estimation, and numerical approximation of complex systems.

## Mortgage Calculator

Description:
Financial modeling tool for computing mortgage payments, amortization schedules, and loan sensitivity to interest rate changes.

## Options Trading Models

Description:
Implementation of derivatives pricing models including Black–Scholes, volatility calculations, and options analytics.

## Trading Engine

Description:
